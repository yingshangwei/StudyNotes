# StudyNotes

对学习过程中的一些关键知识进行记录

# 提示

这里大多数的学习笔记，相比于很多别的博客的学习笔记来说，更注重对其中知识的思考，而并非对知识框架的总结。

因此大多条目都比较松散，也可能没啥上下文关系。它无法帮你归纳知识，但是在你学完和我同样的章节后，看一下我的笔记，或许能给你带来一些不一样的思路。

本人更关注于学习知识过程中知识的推导所带来的乐趣，而并非知识的积累，若是你的目的是积累知识，那这里的笔记应是没有太大帮助的。
